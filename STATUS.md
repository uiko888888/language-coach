# Implementation Status

本文件描述当前代码真实具备的能力，不以路线图或页面入口代替完成状态。

## 已可用

- 本地 Python API、SQLite 数据库和左右布局 Web 前端
- RSS 文章池、来源/主题/内容类型筛选和每日推荐基础
- RSS 摘要与完整正文状态、原始来源链接和手动补充正文
- 文章阅读、可选译文、点词查询和语境生词保存
- 英文/中文/词形/词根词缀混合查询的示例词典
- 一文生成阅读、主旨、改写、选词和首字母基础练习
- 答题记录、错题讲解、同类题和 XP 基础进度
- IELTS、TOEFL、四级、六级、考研、专四、专八、GRE、GMAT 风格配置
- Chrome/Edge 扩展的划词、翻译、保存和网页正文导入
- 来源注册表、来源订阅和 5/15/30 分钟今日内容
- 文章一键翻译、逐段双语对照和手动译文保存
- 单词/短语生词区分和 B1 默认等级下的陌生词初筛
- 兴趣/备考模式切换、选择持久化和两套今日推荐策略
- DeepL 与自托管 LibreTranslate 的私密本地配置入口
- 未收录单词/短语的可操作查询结果、个人语境、翻译和欧路外链
- 生词本返回查询/原文及重复保存更新机制
- Open English WordNet 2025 本地导入、英文义项、例句和语义关系
- WordNet 义项/例句按需中文翻译、逐条对齐、本地缓存和 DeepL 密钥可用性提示
- 查词自动补充中文、浏览器历史返回和练习中心题目自动载入/生成
- WordNet 单词直译、近反义词中文、个人语境搭配和词组独立展示
- 练习中心模式、考试题型选择和当前文章出题入口

## 部分完成

- **考试训练：** 当前是共享生成引擎加独立考试配置，不是完整官方题型、套卷或真题系统。
- **双模式推荐：** 当前会按兴趣/备考模式改变素材排序和今日任务，但尚未使用动态用户画像、目标日期或弱项数据。
- **错题讲解：** 当前是规则驱动讲解，尚未接入有证据校验的 AI 教练。
- **词典：** 已有 Open English WordNet 语义底座和小规模人工词条；中文释义、词源、现代搭配和高频排序仍待 Kaikki/Tatoeba/wordfreq 分层接入，不是欧路级词典库。
- **翻译：** 支持 DeepL 与自托管 LibreTranslate 一键分段翻译和本地缓存；DeepL 会验证密钥并报告拒绝原因，托管服务仍需用户自己的有效密钥或服务地址。
- **陌生词判断：** 当前使用 B1 默认值、词典等级和生词记录，尚未接入动态画像。
- **内容更新：** 用户点击刷新时更新；尚无后台定时任务、失败重试队列和来源健康面板。
- **进度：** 有 XP、连续天数和正确数，但没有分项能力曲线、可信考试预测或周报。

## 当前工程风险

- 后端业务集中在单个 `server.py`，继续扩展前需要逐步拆分模块。
- 文章推荐和题目生成仍以启发式规则为主，质量上限有限。
- RSS、商业媒体、流媒体和字幕具有不同版权边界，不能统一按全文抓取处理。
- 本地 SQLite 适合单用户 MVP，不适合作为未来多用户云服务的最终存储方案。
- 当前测试覆盖后端主流程，但缺少真实浏览器交互覆盖。

## 最近优先级

1. 更多导入方式和用户合辑。
2. 开放词典/语境数据导入、来源标注和高频义项排序。
3. 首次画像、目标与双模式联动。
4. 来源稳定性、去重、备份和前端端到端测试。
5. 正式考试引擎、AI 校验和听力工作台。
