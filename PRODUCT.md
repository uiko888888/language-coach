# Language Coach Product Definition

## 产品定位

Language Coach 是一个以真实内容为素材、以动态用户画像为核心，同时服务于英语兴趣输入和 IELTS/TOEFL 备考的私人 AI 语言教练。

它不是单纯的词典、背单词工具或题库。产品的核心价值是把用户真正愿意阅读和收听的内容，转化为可理解、可练习、可复习、可追踪的个人学习材料。

## 核心用户

当前首先服务单用户、本地优先的个人学习场景：

- 英语约 B1-B2，希望快速提升阅读、听力、词汇运用和考试表现。
- 主要考试方向为 IELTS 和 TOEFL。
- 喜欢英美剧、明星访谈、欧美文化、新闻、小说和博客。
- 能通过重复记住词义，但需要更多语境、搭配和使用时机。
- 阅读理解通常强于听力、口语和选词运用。

法语和西班牙语暂不作为近期主线，但数据模型应保留多语言扩展能力。

## 两种学习模式

### 兴趣模式

用户自由选择或订阅字幕、明星访谈、新闻、小说、博客等内容板块。系统提供轻量任务和成长奖励，但不持续制造考试压力。

### 备考模式

用户选择 IELTS 或 TOEFL，按考试、科目、题型和能力弱点训练。系统保留原文证据，记录错因，并安排相似题和复习任务。

两种模式共享同一个画像、生词本、词块库和能力记录。兴趣模式中的查词、听力和阅读表现可以帮助校准备考画像，但不能直接虚增考试预测分数。

## 首次画像入口

首次使用分为两条路径。

### 已有成绩

适合已经参加过 IELTS、TOEFL、EF SET 或其他测试的用户。填写：

- 测试名称与日期
- 总分和分项成绩
- 目标分数与计划日期
- 每日可投入时间
- 自认为最薄弱的部分

测试类型支持 IELTS、TOEFL、EF SET 和其他测试。总分与分项成绩均可选填，但至少填写一项有效成绩。已有成绩作为画像初始锚点，成绩日期越久，画像置信度越低，后续通过训练持续校准。

### 暂无成绩

适合兴趣学习或不了解当前水平的用户。可以：

- 填写自评水平和兴趣
- 完成 10-15 分钟快速自适应测试
- 暂时跳过测试，先使用再逐步建立画像

快速测试参考 EF SET 的清晰、低负担体验，但使用自主题目和评分逻辑，不复制其受保护内容或品牌表现。

## 动态用户画像

画像包含：

- CEFR 综合等级，以及阅读、听力、词汇、写作、口语分项
- IELTS/TOEFL 当前成绩、目标和考试日期
- 阅读速度、正确率、听力识别率和题型表现
- 证据定位、主旨概括、推断、同义替换、词汇语境、篇章结构、作者态度等能力
- 陌生词、认识但不会用的词、搭配薄弱和听不出来的词
- 内容兴趣、订阅、难度偏好和主动反馈
- 每日投入、连续学习、复习完成度和阶段趋势

画像在有效训练后小幅更新，每 7 天生成总结，每 30 天建议进行局部复测。用户始终可以查看、修改或纠正画像。

能力分和成长 XP 必须分开：能力分反映表现，XP 反映投入和完成度。

## 内容订阅中心

兴趣内容作为一级板块，支持选择、订阅和取消订阅：

- 影视字幕
- 明星与人物访谈
- 国际新闻与时政追踪
- 小说与公共领域文学
- 博客与文化评论
- 科学、技术、商业和健康内容

流媒体内容同时按内容形态和时长组织：

- 经典剧集与电影片段
- 访谈、脱口秀和传统综艺
- 流媒体原创真人秀与综艺
- 短视频与社交媒体片段

```text
微片段     15-60 秒
短片段      1-5 分钟
完整单元    5-20 分钟
长内容     20 分钟以上
```

订阅首先进入应用内“今日内容”和每日推荐。浏览器徽标或通知在后续版本提供，并且必须由用户主动开启、可调整频率、可随时关闭。

内容必须记录来源、作者、原始地址、发布时间、内容类型和完整性。新闻报道、观点评论、学术解释、机构公告和文化评论不能混为一类。

受版权保护的内容优先保存标题、合法摘要、短语境和原始链接；完整字幕、小说或正文仅接受用户合法导入、开放许可或公共领域来源。

## 一份素材多次利用

每份可用素材进入统一处理流程：

```text
采集与来源校验
-> 正文/字幕分段和清理
-> 语言、主题、难度和内容类型识别
-> 生词、词块、重点句和同义替换提取
-> 阅读、听力、填空、词汇和复述练习
-> 作答、讲解和错因记录
-> 相似题与间隔复习
-> 用户画像更新
```

可生成的学习方式包括：

- 原文阅读与可选翻译
- 逐句或逐词听力
- 单词、词块、搭配和重点句提取
- 阅读理解、判断、匹配、摘要填空和选择题
- 选词填空与首字母填空
- 听写、复述和影子跟读
- 错题讲解和同考点巩固

每份素材默认展示系统推荐的学习组合，同时允许用户自由选择阅读、听力、词汇、填空、复述等单项模式。

## 题目拟合与质量

素材语言难度和题目难度是两个独立维度。系统不得因为素材轻松就只能生成简单题，也不得为了提高难度故意制造偏题、怪题或模糊答案。

考试拟合使用题型规则和能力标签，不复制官方或商业题目文本。生成流程至少包含：

```text
素材分析
-> 练习规划
-> 题目生成
-> 答案与原文证据校验
-> 干扰项和难度校准
-> 发布或拒绝
```

题目质量需要检查：

- 是否符合 IELTS/TOEFL 对应题型结构
- 是否存在唯一或规则允许的明确答案
- 是否能定位原文证据
- 干扰项是否合理但不刻意刁钻
- 同义改写是否自然且保持原意
- 推断距离是否符合目标难度
- 是否出现超出素材的信息或极端解释

任何无法通过答案唯一性、证据完整性或自然度检查的题目都不应展示。AI 生成题必须标记为模拟题，系统不得把生成内容冒充官方真题或来源原文。

## 产品原则

1. 原文优先，翻译按需出现。
2. 词汇必须回到语境、搭配和使用条件。
3. 用户可以自由选择，系统推荐不能变成强制路径。
4. 兴趣与备考共享数据，但能力评估必须可信。
5. AI 负责解释、适配和生成，不替代来源、答案规则和质量校验。
6. 每项内容都保留来源与版权边界。
7. 每个版本必须有范围、验收标准、更新日志、提交和标签。
8. 借鉴成熟机制，但不复制受许可证或版权限制的业务代码和题目内容。
