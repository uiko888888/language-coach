## Complete the Words 卡片回顾边界

该模式是“已做题目的回顾”，不是普通生词卡，也不是把首字母填空加入所有考试。

```text
题目来源
-> 历史错题 / 全部已做原题 / 收藏 / 经规则验证的潜在考点
-> 显示原句、已给前缀和空缺
-> 用户输入完整单词
-> 翻卡显示答案、词性、当前语境中文义、完整原句和来源
-> 记录拼写、词形、上下文推断错误
-> 交给 FSRS 安排下一次回顾
```

实施约束：

1. 只用于确实包含 `complete-words` 的考试配置或用户主动选择的词汇训练。
2. “潜在考点”必须通过词形、上下文唯一性和词典覆盖校验，不能把任意长词自动挖空。
3. 原题、模拟题和用户素材分别标注来源；不得导入来源不明的商业题库。
4. 中文义项按当前语境展示，不用一个中文词覆盖所有词性和义项。
5. 卡片评分沿用 FSRS，但拼写表现只更新词汇分项，不直接改变综合 CEFR。

## 调研缺口

- EnglishLearner 的准确网址、产品归属和可访问页面尚未核验。
- LingoLeap 当前只有两张用户截图证据，无法判断其题目生成、排期和统计算法。
- 欧路、Quizlet、Anki、LingQ、Language Reactor 等仍需按同一任务实际走完，而不是只比较功能列表。
- 付费、登录后和移动端流程尚未形成可重复的逐步截图审计。

后续竞品研究统一记录：入口、完成一个任务所需步骤、反馈时机、数据留存、退出/恢复、键盘效率、无障碍、版权边界，以及 Language Coach 是否采用。
